Install Album Server (for Windows)¶

1. Download Winserv_install.msi, or check lastest version here.

  1. Run the program

  2. Set the service address.

The bind field should be the IP address you’d like the clients accessible throw network. A public IP address can be useful for visiting from internet.

Save the configuration, then click start to install and run the service.

Troubleshootings¶

For Windows with Firewall enabled, please make sure your filewall settings allow the port you configured.
