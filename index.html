About Semantic-*

The semantic-* is cross plateform enterprise B/S application framework. It includes 3 major modules working together to be used as a web application of which the server running on servlet 3.0.

There multiple clients. For quick start, it's better starting from the js EasyUI client, see jclient/js/EasyUI quick start.

It includes:

  • The semantic-transact, a structured API for building SQL.
  • The semantic-DA, a JDBC based datasource access layer, managing JDBC database connections and handling typical semantics data processing pattern.
  • The semantic-jserv, a servlet server framework for implementing web applications. It provides a protocol layer using json as data package as well as parsers currently based on google Gson which will be upgraded / replaced in the future.

The semantic-* also including 1 extension module providing some basic workflow engine function, which can be configured with:

  • business table & node instance meta data
  • workflow nodes and commands for stepping to next nodes
  • timeout time
  • event handler's class name for step events and timeout events
  • node command's right based on role, bridging by a configured dataset
  • tasks' and task details' semantics that the engine will create / update data but don't handle the structer directly

In addition to the library and extensions, there is also a maven sample project, jserv-sample, showing how to use sematnic-*.
It's recommended clone and use this project as a web server project template. It should be a good starting point.

About jclient/*

jclient/* is a collection of a few clients that working with semantic-* sample project, the jserv-sample.

It includes:

  • The java client
  • The js client, including an EasyUI template project and a in-developing template project based on Vue.js.
  • The c# client in developing
