Class: Obj3

XComponent.Obj3()

new Obj3()

Main scene Object, like THREE.Mesh, THREE.Points, etc.

Typically, if Visual.vtype is mesh, sys.Thrender will create a THREE.Object3D according to this component.

See also xv.ecs.sys.Thrender.createObj3s.
